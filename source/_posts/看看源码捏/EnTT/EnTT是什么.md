# EnTT 是什么
**EnTT**是一个仅头文件、小巧、易于使用、由现代 C++ 编写的 ECS 库。

使用这个库的最具代表性的项目便是 [**Minecraft**](https://minecraft.net/en-us/attribution/)。

GitHub 地址: https://github.com/skypjack/entt
